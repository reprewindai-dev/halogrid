## Core Message Pillars

- CO2 Router is not an advisory dashboard. It is a deterministic execution control plane.
- The system acts before execution, not after the fact.
- Multi-objective enforcement matters: carbon, water, latency, cost, and policy.
- The decision frame is governed before workload admission.
- The binding decision is explicit and limited to a controlled action set.
- Proof attachment and replayable artifacts make the system auditable.
- Traditional schedulers, carbon APIs, and cloud policies are partial systems, not full execution authority.
- Environmental execution authority is the differentiator.
- Deterministic proof infrastructure creates credibility, replayability, and governance integrity.
